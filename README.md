# Energy Consumption Optimization Model for Factories ⚡

This project presents an AI-based system designed to analyze, predict, and optimize energy consumption in industrial environments. The platform uses machine learning, anomaly detection, and digital twin simulation to help factories improve energy efficiency and reduce operational costs.

---

# Project Overview

Industrial facilities consume large amounts of energy during daily operations. Inefficient machine usage and unexpected faults can lead to energy waste and higher costs.

This system provides a smart dashboard that helps monitor energy usage, forecast future consumption, detect abnormal machine behavior, and simulate different factory operating scenarios to identify more efficient configurations.

---

# Key Features

### Executive Energy Dashboard

Displays overall factory energy consumption, CO₂ emissions, and operational metrics.

### AI Energy Forecasting

Uses a Random Forest model to predict energy consumption, carbon emissions, and estimated electricity costs.

### Factory Digital Twin Simulator

Simulates different operating conditions such as machine load, shift duration, and number of active machines to estimate energy usage and efficiency.

### Machine Anomaly Detection

Uses the Isolation Forest algorithm to detect abnormal machine behavior from IoT sensor data.

### NLP Maintenance Log Classifier

Classifies maintenance logs into High Priority, Routine, or Spam using a simple natural language processing model.
